This is a simple website that explains **math formulas and shortcut methods** in an easy, clean format. It includes algebra formulas, matrix formulas, basic trigonometry rules, and quick-solving tricks. The website layout was created using **Medo AI**, and the explanations were refined using **ChatGPT**.
